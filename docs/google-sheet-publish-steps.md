## 欄位順序

學生端網頁會依照以下欄位讀取資料：

```tsv
日期	開始時間	結束時間	義式組狀態	義式組名額	感官手沖組狀態	感官手沖組名額
```

日期欄可填完整日期，例如 `2026-06-19`，也可填簡短日期，例如 `6/19`。簡短日期會由網頁自動補成今年的完整日期。

狀態欄可填：

- 顯示名額
- 不顯示名額
- 不開放

## 取得可給網頁讀取的 CSV 連結

1. 打開 Google Sheet：教室練習時段。
2. 點上方選單「檔案」。
3. 點「共用」或「分享」相關選項中的「發布到網路」。
4. 在發布設定中選：
   - 分頁：工作表1
   - 格式：逗號分隔值（.csv）
5. 按「發布」。
6. 複製 Google 提供的 CSV 連結。
7. 把連結貼回 Codex，我會把它接到 `app/js/config.mjs`。

目前已完成：這份 CSV 連結已接到 `app/js/config.mjs`。

## 注意

這個 CSV 連結會讓知道連結的人讀取表格內容，所以這張 Sheet 不要放學生個資、電話、付款資訊或私人備註。

## Apps Script 輔助工具

已準備好 Apps Script 程式：

```text
docs/google-sheet-apps-script.js
```

貼上位置：

1. 打開 Google Sheet：教室練習時段。
2. 點「擴充功能」。
3. 點「Apps Script」。
4. 把 `docs/google-sheet-apps-script.js` 的內容完整貼上。
5. 按儲存。
6. 回到 Google Sheet，重新整理頁面。

重新整理後，上方會多一個選單：

```text
練習時段工具
```

功能包含：

- `套用時間下拉選單`：在 `開始時間`、`結束時間` 欄位加入常用時間選項。特殊手動時間會保留，但不顯示無效警告。
- `複製最新一週到下一週`：找表格中日期最新的那一週，整週複製到下一週，日期自動加 7 天。若下一週已經有資料，會停止，避免重複。
- `排序並檢查錯誤`：依照日期、開始時間排序，並把錯誤欄位標紅。
- `封存過期時段`：把本週一以前的資料移到 `歷史資料` 分頁，執行前會先跳出確認。

第一次點選工具功能時，Google 可能會要求授權。這是讓這段 Apps Script 可以讀寫這份 Google Sheet 的必要步驟，不需要按右上角「部署」，也不會產生費用。

輸入輔助：

- 開始時間下拉選項：`8:30`、`10:00`、`13:00`、`14:30`。
- 結束時間下拉選項：`11:30`、`16:00`、`17:30`。
- 下拉選單以外的特殊時間仍可手動輸入，該格會自動取消下拉驗證，所以不會顯示無效警告。
- 若特殊時間那一格之後想恢復下拉選單，重新點一次 `套用時間下拉選單` 即可。
- 日期欄輸入 `619` 或 `0619`，會自動轉成今年的 `yyyy-mm-dd`。
- 開始時間、結束時間輸入 `830`，會自動轉成 `8:30`。
- 開始時間、結束時間輸入 `1500`，會自動轉成 `15:00`。

注意：這類內部自動化不需要按 Apps Script 右上角的「部署」。
